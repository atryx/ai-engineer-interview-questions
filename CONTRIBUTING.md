# Contributing

Thanks for contributing! This repo covers **AI engineer interview questions** — LLMs, RAG, agents, embeddings, evaluation, and production ML systems.

## How to Contribute

1. Fork this repo
2. Add a new question or improve an existing answer
3. Follow the format:
   - **Question** as asked in real interviews
   - **Answer** with depth appropriate to the level
   - **Code examples** where applicable
4. Submit a PR

## Guidelines

- Focus on practical, applied AI engineering (not pure ML research)
- Include the difficulty level (Junior / Mid / Senior / Staff)
- Code examples should be in Python
- Reference real tools and frameworks (LangChain, LlamaIndex, vLLM, etc.)
